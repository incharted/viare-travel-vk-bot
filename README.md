# VIARE Travel VK Bot

VK-бот для туристического сообщества `VIARE Travel`. Проект помогает пользователям подбирать туры внутри VK, а менеджеру и администратору - обрабатывать заявки, отвечать клиентам и управлять контентом без ручного редактирования базы данных.

## Что умеет бот

- Показывает главное меню с разделами: подбор тура, все туры, FAQ, контакты, оплата и бронирование.
- Подбирает тур по сценарию: `По России / За границу`, направление, месяц или точные даты, количество туристов, бюджет, тип отдыха.
- Показывает каталог туров и карточки с описанием, ценой и, при наличии, фото.
- Сохраняет заявки пользователей в SQLite.
- Уведомляет менеджера о новых заявках.
- Позволяет менеджеру отвечать клиенту прямо через бота.
- Поддерживает админ-панель: заявки, фильтры, статусы, экспорт, управление турами, редактирование текстов, рассылка.

## Возможности панели

### Менеджер

- Просмотр списка заявок
- Фильтры по статусам: `Новые`, `В работе`, `Ждут клиента`, `Закрытые`
- `Следующая заявка`
- `Карточка заявки`
- `Ответ по заявке`
- `Ответ по VK ID`
- `Закрыть заявку`

### Администратор

Администратору доступны все возможности менеджера, а также:

- `Статистика`
- `Пользователи`
- `Выгрузка заявок` в `CSV/XLSX`
- `Туры админ`
- Добавление, отключение, включение и изменение цены тура
- Редактирование текстов `FAQ`, `Контакты`, `Оплата`
- `Рассылка`

## Подготовка к запуску

### 1. Установите зависимости

Windows:

```powershell
.\setup_bot.bat
```

или:

```powershell
.\scripts\setup.ps1
```

### 2. Создайте локальный `.env`

Скопируйте `.env.example` в `.env` и заполните нужные значения:

```env
VK_TOKEN=your_token_here
ADMIN_IDS=123456789
MANAGER_IDS=123456789
```

### 3. Запустите бота

```powershell
.\run_bot.bat
```

или:

```powershell
.\scripts\run.ps1
```

## Настройка VK

Для корректной работы нужно:

1. Создать или открыть сообщество VK.
2. Включить сообщения сообщества.
3. Включить Long Poll API.
4. Создать ключ доступа сообщества.
5. Вставить этот ключ в локальный `.env` в переменную `VK_TOKEN`.

Лучше всего отправлять пользователям ссылку формата:

```text
https://vk.me/короткое_имя_сообщества
```

Так пользователь попадает именно в личные сообщения сообщества, а не в случайную беседу.

## Безопасность токена

Токен **нельзя коммитить в GitHub**.

Правильная схема:

- токен хранится только в локальном файле `.env`
- в репозиторий добавляется только `.env.example`
- файл `.env` должен быть в `.gitignore`

Если токен уже где-то засветился публично, его лучше сразу перевыпустить в настройках VK и заменить локально.

## База данных

Проект использует SQLite. В базе хранятся:

- пользователи
- заявки
- туры
- логи
- текстовые блоки контента

Файл базы создается автоматически при первом запуске.

## Экспорт данных

Из панели администратора можно выгружать заявки:

- `CSV`
- `XLSX`

Файлы экспорта сохраняются локально в папку `data/exports/`.


## Краткое описание для учебной работы

В проекте реализован VK-бот туристического сообщества `VIARE Travel`, который автоматизирует первичную работу с клиентами. Бот помогает подбирать туры по направлению, бюджету, датам, количеству туристов и типу отдыха, сохраняет заявки в базе данных и передает их менеджеру. Для администратора реализована панель управления с просмотром статистики, пользователей, заявок, экспортом данных, управлением турами и редактированием контента. Проект использует базу данных SQLite, сценарии общения на основе состояний и интеграцию с VK API через `vkbottle`.
