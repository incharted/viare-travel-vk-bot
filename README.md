# VIARE Travel VK Bot

Проект представляет собой VK-бота для туристического сообщества `VIARE Travel`. Бот автоматизирует первичную работу с клиентами: помогает подобрать тур, сохраняет заявки, передает обращения менеджеру и предоставляет административные инструменты для управления турами и контентом.

## О проекте

Бот разработан как прикладное решение для туристического сообщества во ВКонтакте. Пользователь может взаимодействовать с ботом через сообщения сообщества, получать информацию о турах, подбирать подходящие варианты по заданным параметрам и оставлять заявку на связь с менеджером.

Проект ориентирован не только на пользовательскую часть, но и на внутреннюю работу менеджера и администратора. Для этого реализованы отдельные сценарии обработки заявок, фильтрация обращений, экспорт данных и инструменты для редактирования содержимого бота без изменения кода.

## Стек технологий

- Python 3.11+
- `vkbottle`
- `aiosqlite`
- `python-dotenv`
- `openpyxl`
- SQLite

## Основные возможности

- Главное меню внутри VK
- Подбор тура по сценарию
- Каталог туров
- Сохранение заявок в базе данных
- Передача заявок менеджеру
- Ответ клиенту напрямую через бот
- Админ-панель для управления проектом
- Экспорт заявок в `CSV` и `XLSX`
- Редактирование текстовых блоков без изменения исходного кода

## Пользовательский функционал

Бот поддерживает удобный сценарий подбора тура. Пользователь может:

- выбрать формат поездки: `По России` или `За границу`
- выбрать направление
- указать месяц поездки или точные даты
- указать количество туристов
- указать бюджет
- выбрать тип отдыха
- подтвердить сформированную заявку

Также в боте доступны:

- просмотр всех туров
- FAQ
- контакты
- информация об оплате и бронировании
- кнопка связи с менеджером

## Функционал менеджера и администратора

Для сотрудников реализована отдельная рабочая панель.

### Менеджер

- просмотр списка заявок
- фильтрация заявок по статусам
- переход к следующей заявке
- просмотр карточки заявки
- ответ по номеру заявки
- ответ по `VK ID`
- закрытие заявки

### Администратор

Администратор имеет доступ ко всем возможностям менеджера, а также к дополнительным функциям:

- просмотр статистики
- просмотр пользователей
- выгрузка заявок
- управление каталогом туров
- изменение текстов `FAQ`, `Контакты`, `Оплата`
- рассылка сообщений пользователям

## Работа с заявками

В проекте реализована система заявок с поддержкой статусов:

- `Новая`
- `В работе`
- `Ждет клиента`
- `Закрыта`

Каждая заявка сохраняется в базе данных и может быть обработана через панель сотрудника.

## База данных

Проект использует SQLite. В базе данных хранятся:

- пользователи
- заявки
- туры
- логи
- текстовые блоки контента

Это позволяет не только выдавать ответы пользователю, но и хранить историю взаимодействия, учитывать обращения и администрировать содержимое проекта.

## Запуск проекта

### Windows

Подготовка:

```powershell
.\setup_bot.bat
```

Запуск:

```powershell
.\run_bot.bat
```

### Ручной запуск

```powershell
pip install -r requirements.txt
python main.py
```

## Настройка

Для запуска необходимо создать локальный `.env` на основе `.env.example` и указать:

- `VK_TOKEN`
- `ADMIN_IDS`
- при необходимости `MANAGER_IDS`

## Практическая ценность проекта

Проект можно использовать как основу для автоматизации общения с клиентами в туристическом сообществе VK. Бот помогает структурировать обращения, сократить объем ручной переписки на первом этапе и дать менеджеру готовый инструмент для обработки заявок.

Кроме пользовательского сценария, в проекте реализованы элементы внутренней CRM-логики: учет заявок, статусы, ответы менеджера, экспорт данных и редактирование контента через админскую панель.
